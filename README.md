# rotracker
Команды консоли (Синтаксис):
Добавить блок: Просто введи время и текст. Пример: 14:30 ЗВОНОК КЛИЕНТУ
Добавить подзадачу: Знак плюса, ID родителя и текст. Пример: + 05 Собрать логи
Выполнить/Отменить: Просто введи ID задачи или подзадачи. Пример: 05 или 05-1
Удалить (Remove): Команда rm и ID. Пример: rm 05
Извлечь дамп: Введи export
Интегрировать дамп: Введи import
